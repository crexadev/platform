# CREXA

CREXA is a web platform dedicated to AI-generated music.

## Status

Early development. The application foundation is in place; product features are not yet implemented.

## Foundation

- Next.js App Router
- React
- TypeScript
- Tailwind CSS
- pnpm

## Prerequisites

- Node.js 20.9 or later
- pnpm

## Local development

All Git and application commands must be run inside the `platform/` directory.

```bash
cd platform
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Available commands

| Command | Description |
|---|---|
| `pnpm install` | Install dependencies |
| `pnpm dev` | Start the development server |
| `pnpm lint` | Run ESLint |
| `pnpm build` | Create a production build |
| `pnpm start` | Serve the production build |

## External services

No external services (databases, authentication, storage, payments, or analytics) are configured yet.
